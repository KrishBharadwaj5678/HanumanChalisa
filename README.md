# 🕉️ Hanuman Chalisa App

A simple and devotional **Flutter application** that allows users to read the sacred **Hanuman Chalisa** on their mobile devices.
The app is designed to provide a clean and distraction-free experience for devotees.

![HanumanChalisaDemo](https://github.com/KrishBharadwaj5678/HanumanChalisa/raw/main/HanumanChalisaDemo.jpeg)

## 📱 Features

| Feature                 | Description                                            |
| ----------------------- | ------------------------------------------------------ |
| 📜 Hanuman Chalisa Text | Displays the complete Hanuman Chalisa                 |
| 🎨 Clean UI             | Simple and user-friendly interface                     |
| 📱 Mobile Friendly      | Optimized for Android mobile devices                   |
| ⚡ Lightweight App       | Fast loading and smooth performance                    |
| 🙏 Devotional Purpose   | Helps users read Hanuman Chalisa anytime               |

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```
git clone https://github.com/KrishBharadwaj5678/HanumanChalisa.git
```

### 2️⃣ Navigate to the Project Folder

```
cd HanumanChalisa
```

### 3️⃣ Install Dependencies

```
flutter pub get
```

### 4️⃣ Run the App

```
flutter run
```
